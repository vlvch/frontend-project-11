# RSS aggregator
Add RSS feeds and track new posts with updates every 5 seconds.

## System requirements
OS - Windows/macOS/Linux.
npm ^9.8.0
node ^v20.4.0

## Setup 
```bash
make install
```

## Run 
```bash
make build
```
